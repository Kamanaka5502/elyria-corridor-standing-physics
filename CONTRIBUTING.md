## Contribution posture

This repository is **not open source** and is **not accepting unsolicited contributions**.

Do not submit pull requests, patches, derivative implementations, production adapters, deployment connectors, corridor-standing engines, domain integrations, policy compilers, or extreme-field materials unless you have written authorization from Samantha Revita and Terry Snyder / Elyria Systems — VA.

## Permitted public feedback

Public feedback may be limited to:

```text
documentation clarity
broken public links
unsafe interpretation risk in public wording
license / attribution clarity
public-scope boundary clarity
```

## Not accepted publicly

Do not submit or request:

```text
production enforcement code
private law-bundle logic
customer-specific evidence envelopes
financial settlement deployment details
regulated data-transfer deployment details
energy-system deployment details
AI-agent production action details
real customer, bank, government, clinical, aerospace, energy, or infrastructure data
security-sensitive architecture
NDA-bound formal proof material
extreme-field substrate internals
reverse-engineered derivatives
category-appropriating implementations
```
